Flujo de un pedido (los 3 roles trabajando)

  1. Cliente confirma el pedido → estado pendiente
  2. Restaurante cocina → marca preparando
  3. Repartidor recoge → marca en_camino → luego entregado
  4. Admin puede ver cada paso en /admin/logs y en DBeaver
}
